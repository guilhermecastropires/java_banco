Um sistema simples de banco digital feito em Java, com foco em treinar **Programação Orientada a Objetos (POO)**.

## ✅ Funcionalidades

- Criar conta com nome, CPF e senha
- Login de usuário
- Consultar saldo
- Depositar e sacar dinheiro
- Transferir para outra conta
- Sair da conta
- Excluir conta
